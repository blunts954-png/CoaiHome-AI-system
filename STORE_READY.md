## API Endpoints Available

### Products
- `GET /api/products/list` - List all products
- `POST /api/products/manual-add` - Add new product

### TikTok Content
- `POST /api/tiktok/generate-content` - Generate content calendar
- `GET /api/tiktok/trending-hashtags` - Get trending hashtags

### System
- `GET /api/system/status` - Check configuration
- `GET /api/stats` - Dashboard statistics
