## The Psychology Behind Your Store

### Color Scheme (Conversion-Optimized)

```
Trust Blue     #2563EB  -> Headers, logos, trust elements
Action Orange  #F97316  -> Buy buttons, urgency, CTAs  
Money Green    #10B981  -> Checkout, savings displays
Luxury Black   #1A1A2E  -> Text, premium feel
Clean White    #FFFFFF  -> Backgrounds
Soft Gray      #F5F5F5  -> Cards, sections
```

**Why These Work:**
- Blue = Trust (used by PayPal, Facebook) - 34% trust increase
- Orange = Urgency without panic - drives action

## Step-by-Step: Apply to Shopify

### Step 1: Access Your Shopify Store
1. Go to https://coaihome.myshopify.com/admin
2. Login with your credentials
3. Navigate to "Online Store" → "Themes"

### Step 2: Add Custom CSS
1. Click "Customize" on your active theme
2. Go to "Theme Settings" (gear icon)
3. Click "Custom CSS"
4. Copy entire contents from:
   `store_design/coaihome-theme.css`
5. Paste into Custom CSS box
6. Click "Save"

### Step 3: Configure Colors
1. In Theme Customizer, go to "Colors"
2. Set exact hex codes:
   - Primary: `#2563EB` (Trust Blue)
   - Secondary: `#F97316` (Action Orange)
   - Success: `#10B981` (Money Green)
   - Background: `#FFFFFF`
   - Text: `#1A1A2E`

### Step 4: Add Your Products

**Option A: Manual Import (Recommended for Now)**
1. In Shopify Admin → Products → Add Product
2. Copy product details from dashboard: http://localhost:8000/products
3. Set pricing as shown in system
4. Upload lifestyle images from CJ Dropshipping

**Option B: CSV Import**
```
1. Export products from dashboard
2. Format for Shopify CSV
3. Import via Shopify Admin
```

### Step 5: Add Trust Bar
1. In Theme Customizer, add "Custom Liquid" section
2. Position: Below header
3. Add this HTML:
```html
<div class="trust-bar">
  <div class="trust-bar-item">
    <span>Secure Checkout</span>
  </div>
  <div class="trust-bar-item">
    <span>Free Shipping Over $50</span>
  </div>
  <div class="trust-bar-item">
    <span>30-Day Returns</span>
  </div>
  <div class="trust-bar-item">
    <span>4.9/5 Customer Rating</span>
  </div>
</div>
```
